# 📊 台灣受薪族退休金試算系統 (Taiwan Retirement Calculator)
> 專為台灣受薪階級設計的動態退休現金流模擬工具，幫助你將未來的「財務焦慮」轉化為清晰的「數據藍圖」。

🔗 **[點此立即試用工具](https://williamzou0812.github.io/retirement-calculator/)**

## 💡 專案初衷
市面上的退休試算工具往往過於零碎，勞保、勞退需分開計算，且難以直觀感受到「通貨膨脹」對退休金的侵蝕。本專案透過單一網頁應用程式，整合台灣現行退休金制度，並加入動態圖表與壓力測試，讓使用者能一目了然地評估自身的退休準備狀態。

## ✨ 核心功能
* **雙引擎精算**：整合「勞保年金」與「勞退新制（含6%自提彈性）」的試算邏輯。
* **視覺化現金流**：運用動態圖表呈現退休後的資產消耗曲線，告別死板的 Excel 表格。
* **通膨壓力測試**：內建通膨率調整功能，真實還原未來貨幣購買力。
* **多情境比較**：「一次領」與「月領」的跨年度損益黃金交叉點自動計算。
* **彈性擴充模組**：內建原住民專屬的老年給付計算邏輯（可透過設定動態開啟/隱藏）。

## 🛠️ 技術架構
* **前端介面**：HTML5, CSS3 (響應式設計，支援手機/平板瀏覽)
* **邏輯運算**：Vanilla JavaScript
* **資料視覺化**：Chart.js
